The Bodhi Guide to Moksha
Profiles

Profiles are a powerful tool. They control the layout of your Moksha desktop. But you must be careful not to confuse Profiles with Themes. Themes control the colors of your desktop and the appearance of your Gadgets. Profiles control where those Gadgets are positioned and what Theme you have selected.

Profiles are useful because they make it easy to swap quickly between two entirely different layouts. This is especially useful for changing between a single- and a dual-monitor layout, or when you have two people using a computer with the same login, but with different desktop preferences.

Bodhi Linux's Default Profiles

Bodhi Linux ships with just the one Profile called default. When you change your set up, the changes get saved as Bodhi Profile. To revert any changes that you make you can either scratch the profile in the profile settings window, or select the default profile.


Saving a Personalized Profile

Suppose that you have started with Bodhi Linux's default Profile and have modified it by adding few Gadgets and changing the Theme. You really like the setup you now have, but want to try a different set up, it's easy to save the profile. Open the Profile Selector under Main Menu -> Settings -> Settings Panel -> Settings tab -> Profiles. Click the Add button and a pop-up window will ask you for the name you want to use.

You can then switch to another Profile if you like, make modifications to that one and save it as yet another personal Profile. You can modify and save as many personal Profiles as you like.
