Profiles

Profiles are a powerful tool, they control the layout of your Enlightenment (E17) desktop. You must be careful to not confuse the idea of Profiles with Themes. Themes control the colors of your desktop and the appearance of your Gadgets. Profiles control where those Gadgets are positioned and what Theme you have selected.

Profiles are useful because they allow you to quickly and easily swap between two entirely different layout configurations. This is especially useful for changing between a multi-monitor and single monitor layout. Another use is for when you have two people using a computer with the same login, each of whom prefers a different desktop setup.

Bodhi Linux's Default Profiles

Bodhi Linux ships with several different Profiles for you to choose from during installation: Bare, Fancy, Desktop, Laptop, and Tablet/Netbook. Each has been designed for a particular purpose, like hardware type (Desktop, Laptop. Tablet/Netbook), or type of user (Fancy for those who want it all, Bare for those who don't). The selection of Gadgets is the primary method of optimizing for a target use.

Saving a Personalized Profile

Suppose you have started with one of Bodhi Linux's default Profiles and have modified it by adding few Gadgets and changing the Theme. You really like the setup you have, but want to have a look at a different Profile and are afraid of losing your customizations. Don't worry! It's very easy to save it.

Open the Profile Selector under Main Menu -> Settings -> All -> Settings -> Profiles. Click the Add button and a pop-up window will ask you for the name you want to use.

You can then switch to another Profile if you like, make modifications to that one and save it as yet another personal Profile. Each default Profile can have more than one personal Profile built from it, and you can modify and save as many personal Profiles as you like.

This window is the method for switching among Profiles at any time.
