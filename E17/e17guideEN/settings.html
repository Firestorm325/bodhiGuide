The Bodhi Guide to Enlightenment
Settings

Enlightenment (E17) Settings in Bodhi Linux can be accessed via the Settings Panel at Main Menu -> Settings -> Settings Panel. Each of the settings dialogs can also be accessed directly under Main Menu -> Settings -> All.

Note some settings options are only available when certain Modules are loaded.

Setting

Description

Apps

New ApplicationGUI for creating .desktop files which can be added to the Main Menu.
Favorite ApplicationsAdd/Remove entries to the Favorites menu. Also allows changing order of appearance.
IBar Applications Add/Remove applications to the IBar Gadget. Also allows changing the order of appearance.
Restart Applications Select applications to automatically run after restarting E17.
Startup Applications Select applications to automatically run at startup.

Screen

Virtual DesktopsAdjust the number and layout of Virtual Desktops as well as the animation when flipping between them.
Screen Resolution Set the screen resolution.
Screen Lock Set screen lock time as well as login box location, wallpaper, etc.
Screen Saver Enable/Disable X Screensaver, set timer settings, etc.
Power Management Enable/Disable power management, set standby, suspend, and off times.

Windows

Window Display Set window display behavior including moving, resizing, new windows, and shading.
Window Focus Adjust how mouse interaction changes window focus. Advanced options available.
Window Geometry Adjust window resistance to elements on the screen and parameters when moving/resizing windows via the keyboard.
Window Remembers Automatically use most recent window settings for internal dialog or file manager windows next time they are opened. Can remember all or a subset of window settings including size, position, locks, and more. This dialog is primarily to delete window remembers. Window remembers are set using the window menu.
Window Stacking Adjust settings for raising windows as well as allowing windows above fullscreen window.
Window Maximize Policy Set behavior when maximizing a window. Options include Fullscreen, Smart Expansion, Fill available space. Can also set direction of window expansion.
Client List Menu Set how open windows are listed in Client List Menu. Accessed either through Mouse 3/middle-click or Main Menu -> Windows.
Window List Sets behavior of Window List. Typically accessed by pressing Alt+Tab

Menus

Menu SettingsEnlightenment Main Menu settings. Adjust what application info is displayed as well as menu scrolling behavior.

Language

Language Settings Set the locale settings for the environment.
Input Method Settings Adjust settings for various Input Methods which allow users to input symbols not found on their input device. E.g. entering Chinese characters with a Latin keyboard.

Files

File Manager Adjust settings for the Enlightenment File Manager (EFM).
File Icons Set icons used by EFM for different file types.
Places Adjust settings for Places module. Allows access to file manager via Main Menu as well as using a custom file manager (e.g. PCManFM).

Advanced

Performance Fine tune E17 performance. Allows adjusting framerate, font cache, image cache, edje cache, etc.
Engine Enable use of ARGB windows instead of shaped windows.
Search Directories View/Set directory search paths E17 uses to locate various files. E.g. Images, Fonts, Icons, etc.
Battery Meter Control how battery is monitored. Set polling time, low battery alert criteria, etc.
Configuration Panel Enable/Disable E17 configuration dialogs to be accessed from Main Menu -> Settings -> All.
Environment Variables Set custom E17 environment variables.

Look

Wallpaper Change the wallpaper for all desktops or current desktop.
Wallpaper 2 A fancier dialog for changing wallpaper for all desktops or current desktop.
ThemeSwitch Enlightenment themes. Advanced options allow assigning Themes to individual elements.
Colors Assign custom colors to individual Enlightenment elements. Colors depend on Theme capabilities.
Fonts Assign custom font preferences for Enlightenment. Advanced options allow assigning customizations to individual elements.
Borders Set default window border style.
Icon Theme Set the Icon theme to use throughout Enlightenment.
Mouse Cursor Choose between X or Enlightenment style mouse pointer as well as set the size.
Transitions Set the type of transition to use for certain events e.g. changing the background.
Scaling Scale the overall font settings throughout Enlightenment.
Startup Set the splash screen animation to use at boot-up.
Composite Scale Windows Set animation settings for the scale windows action. Scale Windows is an alternative way of navigating between open windows.
CompositeSet Composite Module effects and tune Composite performance.

Input

KeysKey Bindings (Keyboard shortcuts) settings.
Edge Bindings Similar to Key Bindings. Set actions to occur when using the mouse to interact with the edges of the screen.
Mouse Buttons Set actions to bind to mouse buttons. Mouse buttons can be combined with keyboard for additional bindings.
Mouse Select mouse button orientation and mouse acceleration speed.
Hardware Switches Set actions to specialized hardware bindings (ACPI) e.g. pressing the power button or closing the lid to your laptop.
Touch Settings for touchpads.

Settings

Dialogs Set options for dialog windows including if they should behave like normal windows and, if available, whether they should automatically open in Basic or Advanced mode.
ProfilesCreate, delete, and switch between different Profiles.

Launcher

Everything ConfigurationConfigure the Run Everything program including how choices are sorted, view type, size, alignment, and active plugins.
Everything Plugin: Applications Configure terminal and GUI commands for starting applications.
Everything Plugin: Files Set whether to search and/or show recent files or cache. Also provides option to clear cache.
Everything Plugin: Plugins General configurations for plugins to Run Everything.
Everything Plugin: StartSet order of Everything plugins.

Extensions

ModulesLoad/Unload Enlightenment modules.
ShelvesAdd/Delete Shelves and configure settings for individual Shelves.
Mixer Set which mixer to use for sound and open Mixer channel settings.
Pager Configure settings for Pager Gadgets.
Engage Add, delete, and configure instances of the Engage Gadget.
Connection Manager View available networks and configure access settings.
Gadgets Add available Gadgets to the desktop.

System

**These settings require root privilges
NTFS Configuration Tool Enable/Disable write support for NTFS devices.
Software Sources Select what repositories to check for available packages and updates.
LXDM Autologin Setup user for automatic login to bypass initial login screen.
Elementary Configuration Set options for widgets that use the Elementary widget set.
Synaptic Package Manager Install, uninstall, download, or update packages from available repositories.
LXAppearance Customize look and feel of LXDE applications e.g. PCManFM.
