The Bodhi Guide to Enlightenment
Other Settings

This article will discuss some other configuration options that are not directly of and for Enlightenment (E17). Many applications are not configurable by E17 and this section is included for users who would like to change some aspects of them.

Basic Theming

E17 does have some built in functionality to control the look of GTK/QT applications. To access it, go to Main Menu -> Settings -> All -> Look -> Applications. You will see a screen that lists all of the GTK/QT themes you have installed.

Simply select the GTK/QT you wish to use, then click Apply. When you have found a theme you're happy with, click Close.

E17 + GTK/QT Integration

Some E17 themes have a defined GTK/QT theme to match it. This allows for the GTK/QT theme to be switched automatically when the E17 theme is switched (see Themes for more info). To enable this feature, open Main Menu -> Settings -> All -> Look -> Applications and tick the two boxes for Match E17 theme if possible and Enable Settings Daemon.

It should be noted that there are many E17 themes that do not have a pre-defined GTK/QT, meaning that changing the E17 theme will have no effect on the GTK/QT theme.

It is also possible to fully control every aspect of a GTK/QT by installing the application LXAppearance ...

LXAppearance

LXAppearance is not included in a default installation of Bodhi Linux. To install it, run the command

sudo apt-get install lxappearance

Access it by Main Menu -> Applications -> Preferences -> LXAppearance.

Each tab along the top will allow you to control different aspects.

Widget: This section will allow you to choose among a variety of pre-defined GTK/QT themes. GTK/QT themes that you have installed (see the Bodhi Art Site for more - internet connection required) and as well as those installed by default will appear in the list on the left. The area on the right shows a preview of each theme. Under that the font can be selected - this applies only to fonts within a running GTK/QT application and not change the overall system font.

Color: This section allows you to create your own color scheme for GTK/QT applications. The Use customized color scheme button must be checked for the options to be accessible.

Icon theme: Icon sets can be previewed and selected from this screen. Check out the Bodhi Art Site (online link) for more icons!

Mouse Cursor: If you have installed a different mouse cursor it can be changed here. This will only affect the cursor while it is over the area of a running GTK/QT application.

Font: This section will allow you to set some options that will provide font "smoothing".

Other: These options are defined for Window Managers that are built with GTK/QT and will not have any effect on an E17 desktop.
