Profiles

Profiles are a powerful tool. They control the layout of your Moksha desktop. But you must be careful not to confuse Profiles with Themes. Themes control the colors of your desktop and the appearance of your Gadgets. Profiles control where those Gadgets are positioned and what Theme you have selected.

Profiles are useful because they make it easy to swap quickly between two entirely different layouts. This is especially useful for changing between a single- and a dual-monitor layout, or when you have two people using a computer with the same login, but with different desktop preferences.

Bodhi Linux's Default Profiles

Bodhi Linux ships with several different Profiles for you to choose from during installation: Bare, Fancy, Desktop, Laptop, and Tablet/Netbook. Each has been designed for a particular purpose, for example a form factor (Desktop, Laptop or Tablet), or a type of user (Fancy for those who want it all, Bare for those who don't). The selection of Gadgets is the primary method of optimizing for a target use.

Bodhi Linux has provided a variety of Profiles to suit a variety of use cases, but there is no "one size fits all" and each user is free to make changes to the original Profile, for example by adding/removing Gadgets or changing Themes.

Saving a Personalized Profile

Suppose that you have started with one of Bodhi Linux's default Profiles and have modified it by adding few Gadgets and changing the Theme. You really like the setup you now have, but want to take a look at a different Profile. If you are afraid of losing your customizations during the switch, don't worry! It's very easy to save them.

Open the Profile Selector under Main Menu -> Settings -> Settings Panel -> Settings tab -> Profiles. Click the Add button and a pop-up window will ask you for the name you want to use.

You can then switch to another Profile if you like, make modifications to that one and save it as yet another personal Profile. Each default Profile can have more than one personal Profile built from it, and you can modify and save as many personal Profiles as you like.

This window also allows you to switch between Profiles at any time.
